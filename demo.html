jQuery UI Carousel plugin v0.3

Requires jQuery v1.4+ and jQuery UI Widget Factory 1.8+ (now available without core)

Due to overwhelmingly popular demand jQuery Carousel now includes autoScroll feature...okay perhaps not overwhelmingly but it's included as an addon :)

Documentation coming shortly

API

  • .carousel('next') // move to next page
  • .carousel('prev') // move to prev page
  • .carousel('goTo', item) // move to 'item' - item === number, element or jquery obj
  • .carousel('addItems', items) // add items to carousel - items === jquery obj
  • .carousel('option', 'optionName', [value]) // Get or set any carousel option. If no value is specified, will act as a getter.
  • .carousel('option', options) // Set multiple carousel options at once by providing an options object.
  • .carousel('destroy') // reverts carousel back to original state
  • .carousel('start') // starts auto scroll NOTE: requires jquery.ui.carousel-autoscroll.js
  • .carousel('stop') // stops auto scroll NOTE: requires jquery.ui.carousel-autoscroll.js

Note: Options that can be changed after instantiation are currently limited to:

  • itemsPerPage
  • itemsPerTransition
  • noOfRows
  • pagination
  • nextPrevActions
